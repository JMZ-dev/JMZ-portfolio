# WinTech Portfolio (React + Vite)

Stack:
- Three.js (3D background)
- GSAP (intro + scroll reveal)
- Framer Motion (overlay + case-study modal animations)
- Typed.js (typing hero text)

## Run
1) Install Node.js (LTS)
2) In this folder:
```bash
npm install
npm run dev
```
Open the URL shown in the terminal (usually http://localhost:5173)

## Build
```bash
npm run build
npm run preview
```

## Customize
- Edit your contact links in `src/sections/Contact.jsx`
- Update projects in `src/data/projects.js`
- Replace images in `src/assets/` with real screenshots when ready
